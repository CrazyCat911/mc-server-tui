# Minecraft Server TUI

A simple TUI to manage minecraft servers _(currently in development)_

## Getting Started

### Prerequisites

- Recent version of Python
- `pip` (python package manager)
- `pipenv`

### Installation

#### 1. Clone the Repository

```sh
git clone https://github.com/CrazyCat911/mc-server-tui.git
cd mc-server-tui
```

#### 2. Install dependencies

```sh
pipenv install
pipenv shell
```

#### 3. Run the project

```sh
python main.py
```
