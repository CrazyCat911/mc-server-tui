# Minecraft Server TUI

A simple TUI to manage minecraft servers

> [!WARNING]
> This project is currently under active development.
> It is not a production-ready app.

## Getting Started

### Prerequisites

- Python (latest stable version recommended)
- `pip` (python package manager)
- `pipenv`

### Installation

#### 1. Clone the Repository

```sh
git clone https://github.com/CrazyCat911/mc-server-tui.git
cd mc-server-tui
```

#### 2. Install dependencies

```sh
pipenv install
pipenv shell
```

#### 3. Run the project

```sh
python main.py
```
